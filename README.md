This is an implementation of C4.5 algorithm in python.
Creates tree recursively in c45 function and prints tree

Here is an example print for dataset Tennis:

```

`- root - 
   |- outlook - sunny
   |  |- humidity - high
   |  |  `- play - no
   |  `- humidity - normal
   |     `- play - yes
   |- outlook - overcast
   |  `- play - yes
   `- outlook - rainy
      |- windy - false
      |  `- play - yes
      `- windy - true
         `- play - no
```
